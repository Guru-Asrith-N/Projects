## Overview
This project performs data cleaning and exploratory data analysis (EDA) on a student performance dataset to understand factors affecting academic outcomes.

## Dataset
- Source: UCI Machine Learning Repository
- Records student academic, demographic, and behavioral attributes

## Objectives
- Analyze the impact of study time, failures, and absences on final grades
- Identify correlations between academic features

## Key Steps
- Data cleaning (handling missing values & encoding categorical data)
- Exploratory Data Analysis (EDA)
- Visualization of important trends

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
