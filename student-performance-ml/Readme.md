# Student Performance Prediction

## Overview
This project predicts whether a student will pass or fail based on academic and demographic features using supervised machine learning models.

## Dataset
Student Performance Dataset from Kaggle.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Results
Random Forest achieved the best accuracy among the tested models.
